# event-aggregator

A web application that aggregates events from multiple sources (Meetup, Facebook Events, Eventbrite, and local venues) into one centralized platform for your city.

# Features

- **Multi-Source Aggregation**: Collect events from Meetup, Facebook Events, Eventbrite, and local venues
- **Smart Filtering**: Filter events by category, date, source, and price
- **Search Functionality**: Find events by keywords, titles, or locations
- **Event Details**: Comprehensive event information including date, time, location, and descriptions
